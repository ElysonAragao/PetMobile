# **App Name**: PacienteMobile

## Core Features:

- Cadastro de Pacientes: Captura os dados do paciente: ID, Nome, Endereço, Telefone, CPF, Código do Plano de Saúde, Nome do Plano de Saúde.
- Armazenamento de Dados: Persiste os registros dos pacientes para posterior recuperação e gerenciamento.
- Operações de Registro: Permite aos usuários criar novos registros, consultar registros existentes e excluir registros.
- Validação de ID Único: Exibe um aviso quando um usuário insere um ID duplicado.
- Tela de Navegação Principal: Direciona os usuários para as principais funcionalidades do aplicativo (cadastro de pacientes, cadastro de exames, movimentação).
- Cadastro de Exames: Permite o cadastro de diferentes tipos de exames.
- Movimentação de Paciente/Exame: Permite selecionar um paciente e os exames solicitados para ele.
- Geração de QR Code: Gera um ou mais QR Codes contendo as informações do paciente e exames, dividindo-as se excederem o limite de caracteres.
- Compartilhamento de QR Code: Permite o compartilhamento do(s) QR Code(s) via WhatsApp, e-mail, ou impressão em PDF.
- Leitura de QR Code: Permite a leitura do QR Code e transforma as informações em um PDF ou em dados no padrão do sistema da recepção.

## Style Guidelines:

- Cor primária: Azul claro (#ADD8E6) para evocar uma sensação de calma e confiança.
- Cor de fundo: Azul muito claro (#F0F8FF), um tom mais claro do primário, criando um pano de fundo suave e limpo.
- Cor de destaque: Um azul-arroxeado claro (#A0B0FF), um tom frio perto da cor primária, mas com maior contraste, destacando botões e elementos interativos.
- Fonte do corpo e título: 'PT Sans', um sans-serif humanista com um visual moderno e um pouco de calor.
- Use ícones simples e claros para representar ações e campos de dados.
- Uma interface limpa e com guias para listagem de pacientes, cadastro de pacientes, cadastro de exames e movimentação de paciente/exame.
- Transições suaves ao alternar entre as telas de navegação e exibição.
